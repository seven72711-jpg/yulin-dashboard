## 十、门店运转逻辑

```
平台拉新 → 首次体验 → 加微信/留印象 → 储值锁客 → 次月到店 → 指定关系 → 私定提价
    ↓          ↓           ↓           ↓          ↓         ↓         ↓
  43%客源   228均价      存手机号    75.9%完成  22.8%留存   点客率    均价¥450
  抽佣20%   77%下月不来   技师主动    办卡=承诺   复购池    指定客    5人能做
```

**真实复购数据（4月→5月跨月追踪）：**
- 4月605个有手机号的客户 → 5月138人回来 = **跨月留存率22.8%**
- 回头客人均¥477，是新客¥303的1.57倍
- 77%客户下个月不来了

**利润增长的真实路径：**
```
跨月留存22.8%→30% → 复购池扩大 → 指定关系建立 → 私定转化 → 客单价/净利率自然提升
                                                                    ↓
                       同时：复购客走直付 → 拉新需求降低 → 平台费自然缩减
```

### 核心矛盾（4→5月跨月验证）
- **77%客户下个月不再来**。门店月月换血3/4的客源，必须持续拉新才能维持营收
- 回头客人均消费¥477（新客¥303），留住一个老客的价值是新客的1.5倍+
- 11个技师仅4个做私定（占营收9.1%），私定的前置条件是指定关系，而指定靠复购积累
- 平台费高是果，留存率低才是因

### 利润杠杆（4→5月跨月验证）
1. **跨月留存率22.8%→30%**：每月多留≈45人，按人均¥477算≈¥2.1万增量，零获客成本
2. 首次体验→技师连接：加微信、记住偏好、邀约下次到店
3. 复购客→指定客→私定客：逐级转化，客单价从228→340→450

## 二十七、折扣成本归因——门店在替技师买单

**数据来源：5月水单 + 工资表**

- 技师提成按**劳动业绩¥209,054**计算（原价口径）
- 门店实收只有**¥176,759**
- 差额**¥32,295/月** = 平台折扣 + 直付折扣

| 渠道 | 折扣金额 | 谁承担 |
|------|---------|--------|
| 美团 | ¥13,620 | 门店100% |
| 抖音 | ¥6,285 | 门店100% |
| 直付 | ¥12,390 | 门店100% |
| **合计** | **¥32,295** | **门店100%，技师0%** |

**结论：** 技师不管客户用了几折券，照拿原价提成。每月¥32,295的折扣，100%由门店利润承担。年化¥387,540。

**如果改为"按实收提成"（同比例）：**
- 技师工资从¥107,004降至¥90,502
- 月省¥16,500，年省¥198,000
- 净利润从¥17,285翻倍到¥33,787
- **风险：** 技师到手减少约15%，可能抵制/离职


## 二十八、边际利润——每多做1单只赚¥44

保本点¥112,000（约448单）。5月841单，超保本393单。净利润¥17,285 ÷ 393单 = **¥44/单**。

如果超保本的单全是直付全价：边际利润≈¥109/单。实际只有¥44，因为增量单大量是平台折扣单、提成先抽走一半。

**含义：** 靠"多做单"增收的效率很低。每多拉1个美团新客，门店到手只有¥44。
